# CasinoGames


#### CMake
```sh
$ cmake CMakeLists.txt          # generates build configuration (Makefile)
```

#### Make
```sh
$ make              # builds executable based on Makefile
$ make clean_target # delete output files
$ ./CasinoGames     # viola!
```
